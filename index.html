Co potrafię?

» Język Opisowy HTML5 (90%)
Tworzenie struktur bazując na HTML w wersji 5 jest dla mnie tak łatwe jak poruszanie ręką... Każda moja strona jest dopracowana zarówno dla komputerów jak i urządzeń mobilnych.

» Język Opisowy CSS3 (85%)
Tworzenie wyglądu strony to dla mnie prościzna. Do CSS wykorzystuje rozszerzenie o nazwie SCSS - pomaga mi to stworzyć lepszą stronę dla każdego czytelnika ;)

» Język Skryptowy PHP (79%)
Co prawda PHP jest łatwym językiem potrafię w nim dużo ale nie wszystko. Nadal się go uczę aczkolwiek z uśmiechem na twarzy powiem iż nie mam z nim tak dużych problemów jak kiedyś. Tworzenie aplikacji internetowych jest proste. Mało tego napisanie botów na ts3 czy discord też nie jest dla mnie trudne.

» Język Skryptowy JavaScript (81%)
JavaScript aktualnie służy mi tylko do poleszpenia wyglądu strony etc. (tylko do rzeczy związanych z front-endem). W przyszłości planuje rozwinąć swoje wykorzystywanie tego języka także i do front-endu wykorzystując przy tym frame-work NodeJS.

» Zarządzanie systemami Linux (89%)
Śmiało przyznam iż linuxa pierwszy raz uruchomiłem w moje 11 urodziny. Owszem było trudno ale łatwo się do niego przyzwyczaiłem teraz instalacja pakietów lub ogólne administrowanie systemem linux nie jest dla mnie wyzwaniem... Ba! jest dla mnie rozrywką, wszystko idzie łatwo i szybko.
